Github 與 Halloween

今天萬聖節，打開 github 看到這個 style 有嚇一跳

一開始以為 github 在改這部份的 style，看了說明才知這是為了萬聖節的活動，說真的我有被顏色嚇一跳。想到萬聖節就頭痛，以後小孩上幼稚園，我可以不跟進這個活動嗎？

每年 April Fool’s day 都有各大阿宅網站做出惡搞，沒想到連萬聖節都開始有活動了
